# Archivio Progetti Video

Questa cartella contiene tutti i progetti video generati, organizzati per data e argomento.

## Struttura Standard

Ogni nuovo video deve avere la sua sottocartella nominata come segue:
`YYYY-MM-DD_nome_progetto_slug`

Esempio: `2026-02-19_scarpe_bianche`

### Contenuto della cartella progetto:

1.  **`step1_creative_script.md`**: Contiene l'output dell'Agente 1 (Script, Scene, Mood).
2.  **`step2_grok_prompts.md`**: Contiene l'output dell'Agente 2 (Codici tecnici per Grok).

---

## Come salvare i file qui

Se usi Perplexity:
1.  Copia l'output dell'Agente.
2.  Crea un nuovo file qui su GitHub dentro la cartella corretta.
3.  Incolla e salva.
